Refind

Search smarter with shortcuts

A Firefox extension that adds customizable keyboard shortcuts to instantly refine your search queries. Append site filters, keywords, or any text with a single keystroke.

Install for Firefox View on GitHub

Features

⌨

Custom Keyboard Shortcuts

Create your own shortcuts with any modifier combination. Press-to-record makes capturing shortcuts effortless.

🔍

Multi-Engine Support

Works seamlessly on Google, Bing, DuckDuckGo, and Yahoo. Your shortcuts follow you everywhere.

⚡

Instant Refinement

Append any text to your current search query. Filter by site, add keywords, or customize results instantly.

⚙

Full Control

Enable, disable, edit, or delete shortcuts anytime. Warnings for browser shortcut conflicts keep you informed.

How to Use

1

Search as usual

Start your search on Google, Bing, DuckDuckGo, or Yahoo just like you normally would.

2

Press your shortcut

Hit your keyboard shortcut to instantly append text to your search query. The default shortcut adds "site:reddit.com" to find Reddit discussions.

3

Customize to your needs

Click the extension icon to add new shortcuts, change what text gets appended, or create shortcuts for different sites and filters.

Example Shortcuts

Cmd + Shift + E Append "site:reddit.com"
Cmd + Shift + S Append "site:stackoverflow.com"
Cmd + Shift + G Append "site:github.com"
