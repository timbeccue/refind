Privacy Policy

SearchGoggles Browser Extension

Last updated: January 30, 2026

Summary

SearchGoggles does not collect, transmit, or share any user data. All data stays on your device.

Data Storage

SearchGoggles stores the following data locally in your browser:

  • Your custom keyboard shortcuts
  • Associated search filter terms (e.g., "site:reddit.com")
  • Shortcut enabled/disabled states

This data is stored using your browser's local storage API and never leaves your device.

Data Collection

SearchGoggles does not collect:

  • Personal information
  • Browsing history
  • Search queries
  • Usage analytics
  • Any other data

Third Parties

SearchGoggles does not share any data with third parties because no data is collected in the first place.

Permissions

The extension requests the following permissions:

  • Storage: To save your keyboard shortcuts locally
  • Tabs: To modify the current tab's URL when you use a shortcut
  • Host permissions for search engines: To detect when you're on a search results page

Contact

For questions about this privacy policy, please open an issue at github.com/timbeccue/searchgoggles.
